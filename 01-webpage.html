Conditionals

Conditional statements are useful while coding different decisions which lead to different outcomes. “If” statements run a condition only if it is true. An “else if” statement runs an alternative condition if the previous is false. An “else” statement runs yet another condition if the previous were all false.

if(age <= 17){

console.log('Sorry, youre too young to do anything');

} else if(age >= 18 && age < 21){

console.log('Yay! You can vote!');

} else if(age >= 21 && age <25){

console.log('Yay! You can drink');

} else(age >= 25)

console.log('Yay! You can rent a car!');

Objects

An object is a variable which can hold many different properties within them. These may also hold methods which are actions that can be done to the object. These actions are stored functions.

let scott = {

firstName: Scott,

lastName: Billings,

eyeColor: Brown,

age:30,

};

Arrays

An array is a special kind of variable which is a list of ordered items. These Items can hold more than one value and are accessible by their index number.

let scott = ["scott", "Billings", "Brown", 30,];
